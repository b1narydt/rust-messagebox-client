```rust
/// Outgoing WebSocket event envelope.
/// Wire format: {"event": "joinRoom", "data": "room-id"} or
/// {"event": "sendMessage", "data": {"roomId": "...", "message": {...}}}
#[derive(Serialize, Deserialize, Clone, Debug)]
pub struct WsEvent<T: Serialize> {
    pub event: String,
    pub data: T,
}

/// Incoming WebSocket event envelope (deserialized from server frames).
#[derive(Deserialize, Clone, Debug)]
pub struct WsIncomingEvent {
    pub event: String,
    pub data: serde_json::Value,
}

/// Payload for the sendMessage WebSocket event.
#[derive(Serialize, Deserialize, Clone, Debug)]
#[serde(rename_all = "camelCase")]
pub struct WsSendMessageData {
    pub room_id: String,
    pub message: WsSendMessagePayload,
}

/// The message object inside a sendMessage event.
#[derive(Serialize, Deserialize, Clone, Debug)]
#[serde(rename_all = "camelCase")]
pub struct WsSendMessagePayload {
    pub message_id: String,
    pub recipient: String,
    pub body: String,
}

/// Reconnect options matching SDK's WebSocketTransport pattern.
pub struct WsReconnectOptions {
    pub reconnect: bool,
    pub max_retries: u32,
    pub backoff_ms: u64,
}

    fn default() -> Self {
        Self { reconnect: true, max_retries: 5, backoff_ms: 1000 }
    }
```

**3. Create `src/websocket.rs`:**

Build `MessageBoxWebSocket` struct with these fields:
- `tx: mpsc::Sender<String>` — outgoing message queue (serialized JSON strings)
- `connected: Arc<AtomicBool>` — non-blocking connection check
- `subscriptions: Arc<Mutex<HashMap<String, Arc<dyn Fn(PeerMessage) + Send + Sync>>>>` — room_id to callback
- `pending_acks: Arc<Mutex<HashMap<String, oneshot::Sender<bool>>>>` — ack_event_key to oneshot
- `joined_rooms: Arc<Mutex<HashSet<String>>>` — idempotency tracking (Pitfall 6)
- `read_handle: JoinHandle<()>` — for clean shutdown
- `write_handle: JoinHandle<()>` — for clean shutdown

Implement:
- `pub async fn connect(url: &str) -> Result<Self, MessageBoxError>`: Call `tokio_tungstenite::connect_async(url)`, split with `futures_util::StreamExt::split()`. Spawn write task that reads from `mpsc::Receiver<String>` and sends `WsMessage::Text`. Spawn read task that parses `WsMessage::Text` as `WsIncomingEvent`, dispatches:
  - Events matching `sendMessage-{room_id}` pattern: look up subscription callback, invoke it with parsed `PeerMessage`
  - Events matching `sendMessageAck-{room_id}` pattern: look up pending_ack oneshot, send `true` (or `false` for error status)
  - Other events: log and ignore
- `pub fn is_connected(&self) -> bool`: Read `AtomicBool`
- `pub async fn join_room(&self, room_id: &str) -> Result<(), MessageBoxError>`: Check `joined_rooms` HashSet, skip if already joined. Serialize `WsEvent { event: "joinRoom", data: room_id }` and send via `tx`. Add to `joined_rooms`.
- `pub async fn leave_room(&self, room_id: &str) -> Result<(), MessageBoxError>`: Remove from `joined_rooms` and `subscriptions`. Serialize `WsEvent { event: "leaveRoom", data: room_id }` and send via `tx`.
- `pub async fn subscribe(&self, room_id: String, callback: Arc<dyn Fn(PeerMessage) + Send + Sync>)`: Insert into subscriptions map.
- `pub async fn emit_send_message(&self, room_id: &str, payload: WsSendMessageData, ack_key: &str, ack_tx: oneshot::Sender<bool>) -> Result<(), MessageBoxError>`: Register ack_tx in pending_acks under ack_key. Serialize `WsEvent { event: "sendMessage", data: payload }` and send via `tx`.
- `pub async fn disconnect(&mut self)`: Set `connected` to false. Abort `read_handle` and `write_handle`. Drain `pending_acks` — send `false` to all remaining oneshot channels. Clear `subscriptions` and `joined_rooms`.

CRITICAL PITFALLS to honor:
- Pitfall 2: Never hold ws_state Mutex across await. Clone `tx` sender from guard, drop guard, then await send.
- Pitfall 3: Auto-decryption happens in the read task BEFORE invoking callback (not inside callback). The read task needs wallet access for this — but per Research open question 4, pass the wallet to the read task is complex. Instead, the `MessageBoxClient` layer handles decryption before callback invocation. The `MessageBoxWebSocket` dispatches raw `PeerMessage` structs; the client wrapper decrypts.
- Pitfall 6: `join_room` checks `joined_rooms` HashSet before emitting.
- Pitfall 7: Timeout path must remove entry from `pending_acks`.
- Use `WsMessage::Text` for all sends; only parse `WsMessage::Text` on receive.
- Read task should set `connected = false` when the stream ends (connection dropped).

```rust
async fn ensure_ws_connected(&self) -> Result<(), MessageBoxError> {
    let mut guard = self.ws_state.lock().await;
    if guard.as_ref().map(|ws| ws.is_connected()).unwrap_or(false) {
        return Ok(());
    }
    // Build WebSocket URL from HTTP host: replace http(s) with ws(s)
    let ws_url = self.host()
        .replace("https://", "wss://")
        .replace("http://", "ws://");
    let ws = crate::websocket::MessageBoxWebSocket::connect(&ws_url).await?;
    *guard = Some(ws);
    Ok(())
}
```

**3. Add `listen_for_live_messages` public method:**

```rust
pub async fn listen_for_live_messages(
    &self,
    message_box: &str,
    on_message: Arc<dyn Fn(PeerMessage) + Send + Sync>,
) -> Result<(), MessageBoxError> {
    let identity_key = self.get_identity_key().await?;
    let room_id = format!("{identity_key}-{message_box}");

    self.ensure_ws_connected().await?;

    // Wrap the user callback with auto-decryption.
    // Clone wallet + originator for the decryption wrapper.
    let wallet = self.wallet.clone();
    let originator = self.originator.clone();
    let decrypting_callback: Arc<dyn Fn(PeerMessage) + Send + Sync> = Arc::new(move |mut msg: PeerMessage| {
        let w = wallet.clone();
        let orig = originator.clone();
        let cb = on_message.clone();
        // Spawn a task to decrypt then invoke the user callback
        tokio::spawn(async move {
            match crate::encryption::try_decrypt_message(&w, &msg.body, &msg.sender, orig.as_deref()).await {
                Ok(decrypted) => { msg.body = decrypted; }
                Err(_) => { /* plaintext fallback — leave body as-is */ }
            }
            cb(msg);
        });
    });

    // Acquire lock, join room and subscribe, then drop lock
    let guard = self.ws_state.lock().await;
    if let Some(ref ws) = *guard {
        ws.join_room(&room_id).await?;
        ws.subscribe(room_id, decrypting_callback).await;
    }
    Ok(())
}
```

NOTE on Pitfall 3: The CommsLayer callback is `Fn(PeerMessage)` (sync). Decryption requires async. Solution: spawn a tokio task inside the wrapper callback that decrypts then invokes the user callback. This is acceptable because the read task dispatches quickly and the actual processing happens in the spawned task.

**4. Add `send_live_message` public method:**

```rust
pub async fn send_live_message(
    &self,
    recipient: &str,
    message_box: &str,
    body: &str,
) -> Result<String, MessageBoxError> {
    // Encrypt + generate message ID (same as send_message)
    let encrypted = crate::encryption::encrypt_body(self.wallet(), body, recipient, self.originator()).await?;
    let message_id = crate::encryption::generate_message_id(self.wallet(), body, recipient, self.originator()).await?;

    let room_id = format!("{recipient}-{message_box}");
    let ack_key = format!("sendMessageAck-{room_id}");

    // Check if WS is available — clone tx from guard, drop guard before await
    let ws_available = {
        let guard = self.ws_state.lock().await;
        guard.as_ref().map(|ws| ws.is_connected()).unwrap_or(false)
    };

    if !ws_available {
        return self.send_message(recipient, message_box, body).await;
    }

    let (ack_tx, ack_rx) = tokio::sync::oneshot::channel::<bool>();

    // Build payload
    let payload = crate::types::WsSendMessageData {
        room_id: room_id.clone(),
        message: crate::types::WsSendMessagePayload {
            message_id: message_id.clone(),
            recipient: recipient.to_string(),
            body: encrypted,
        },
    };

    // Emit — acquire lock briefly
    {
        let guard = self.ws_state.lock().await;
        if let Some(ref ws) = *guard {
            ws.emit_send_message(&room_id, payload, &ack_key, ack_tx).await?;
        }
    }

    // Await ack with 10-second timeout
    match tokio::time::timeout(std::time::Duration::from_secs(10), ack_rx).await {
        Ok(Ok(true)) => Ok(message_id),
        _ => {
            // Pitfall 7: clean up pending ack on timeout
            let guard = self.ws_state.lock().await;
            if let Some(ref ws) = *guard {
                ws.remove_pending_ack(&ack_key).await;
            }
            drop(guard);
            // Fall back to HTTP
            self.send_message(recipient, message_box, body).await
        }
    }
}
```
