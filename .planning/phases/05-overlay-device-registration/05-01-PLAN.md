Add to the `impl<W: WalletInterface + Clone + 'static> MessageBoxClient<W>` block in `src/host_resolution.rs`:

1. `pub async fn anoint_host(&self, host: &str) -> Result<(), MessageBoxError>`:
   - Get identity key via `self.get_identity_key().await?`
   - Derive public key via `self.wallet().get_public_key(GetPublicKeyArgs { identity_key: false, protocol_id: Some(Protocol { security_level: 1, protocol: "messagebox advertisement".to_string() }), key_id: Some("1".to_string()), counterparty: Some(Counterparty { counterparty_type: CounterpartyType::Anyone, public_key: None }), privileged: false, privileged_reason: None, for_self: Some(true), seek_permission: None }, self.originator()).await`
   - Build PushDrop fields: `[hex::decode(&identity_key)?, host.as_bytes().to_vec()]`
   - Call `build_pushdrop_lock_from_fields_and_pubkey(&fields, &pk_result.public_key.to_der())`
   - Call `self.wallet().create_action(CreateActionArgs { description: "Anoint host for overlay routing".to_string(), input_beef: None, inputs: vec![], outputs: vec![CreateActionOutput { locking_script: Some(locking_script.to_binary()), satoshis: 1, output_description: "Overlay advertisement output".to_string(), basket: Some("overlay advertisements".to_string()), custom_instructions: None, tags: vec![] }], lock_time: None, version: None, labels: vec![], options: Some(CreateActionOptions { randomize_outputs: BooleanDefaultFalse(Some(false)), accept_delayed_broadcast: BooleanDefaultTrue(Some(false)), ..Default::default() }), reference: None }, self.originator()).await`
   - Extract BEEF from `create_result.tx` (the raw transaction bytes)
   - Broadcast via `TopicBroadcaster::new(vec!["tm_messagebox".to_string()], TopicBroadcasterConfig::default(), LookupResolver::new(LookupResolverConfig { network: self.network.clone(), ..Default::default() }))` then `broadcaster.broadcast(&beef_hex).await`
   - Map errors to `MessageBoxError::Overlay(...)`.

2. `pub async fn revoke_host_advertisement(&self, token: &AdvertisementToken) -> Result<(), MessageBoxError>`:
   - Step 1: `self.wallet().create_action(CreateActionArgs { description: "Revoke MessageBox host advertisement".to_string(), input_beef: Some(token.beef.clone()), inputs: vec![CreateActionInput { outpoint: format!("{}.{}", token.txid, token.output_index), input_description: "Revoking host advertisement token".to_string(), unlocking_script: None, unlocking_script_length: Some(73), sequence_number: None }], outputs: vec![], ... })` to get signable transaction.
   - Step 2: Parse signable transaction, derive sighash preimage for input 0.
   - Step 3: Use `self.wallet().create_signature(CreateSignatureArgs { data: preimage_bytes, protocol_id: Some(Protocol { security_level: 1, protocol: "messagebox advertisement".to_string() }), key_id: Some("1".to_string()), counterparty: Some(Counterparty { counterparty_type: CounterpartyType::Anyone, public_key: None }), privilege: false, privileged_reason: None, for_self: Some(true), seek_permission: None, hash_to_directly_sign: None }, self.originator()).await` to produce DER signature.
   - Step 4: Build unlock script: `<sig_der_bytes + 0x41 sighash byte>` as a single ScriptChunk data push.
   - Step 5: `self.wallet().sign_action(SignActionArgs { reference: create_result.reference, spends: { input 0 -> unlocking_script bytes }, ..Default::default() }, self.originator()).await`
   - Step 6: Broadcast result via TopicBroadcaster same as anoint_host.
   - Map errors to `MessageBoxError::Overlay(...)`.

   **Note on sighash preimage:** This is the trickiest part. The TS SDK's `PushDrop.unlock()` handles sighash internally. In Rust, we may need to parse the `signable_transaction.tx` as a `Transaction`, compute the sighash for input 0 using `SIGHASH_ALL|SIGHASH_FORKID` (0x41), and pass those bytes to `create_signature`. Check if the Rust SDK has a `TransactionSighash` or `SighashPreimage` helper. If not, use `wallet.create_signature` with `hash_to_directly_sign` set to the hash. This is an OPEN QUESTION from research — verify at implementation time against the SDK source.

3. Write unit tests:

   - `test_revoke_host_args_correct`: Verify outpoint format string is `"{txid}.{output_index}"`.

All methods use `map_err(|e| MessageBoxError::Overlay(e.to_string()))` for SDK errors.
